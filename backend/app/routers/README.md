# Plagiarism & AI Detector

Веб-приложение для проверки текстов на плагиат и обнаружения использования искусственного интеллекта.

## Описание

Приложение позволяет:
- Загружать документы в форматах: TXT, PDF, DOCX, DOC, RTF
- Проверять текст на оригинальность (плагиат)
- Определять вероятность использования ИИ
- Сохранять историю проверок
- Скачивать отчеты

## Установка и запуск

### Требования
- Python 3.8+
- pip
